ЗАМЕЧАНИЕ ПО ИМЕНАМ
Имена всех единиц трансляции должны заканчиваться знаком $.
Это необходимо для того, чтобы при запуске TMKMAKE с опцией *EVENTF генерируемые мемберы файла EVFEVENT имели разные имена.
В противном случае мембер с результатами компиляции, например, SRC/FORMAT будет затирать мембер с результатами компиляции TEST/FORMAT.

Данное решение считается временным.
Необходима доработка TMKMAKE в части формирования файла EVFEVENT.
